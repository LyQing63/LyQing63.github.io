leetcode 题421:数组中两个数的最大异或值

发表于2023-11-05|更新于2023-11-06|Algorithm
|字数总计:1.4k|阅读时长:5分钟|阅读量:

题目

给你一个整数数组nums，返回nums[i] XOR nums[j]的最大运算结果，其中0 ≤ i ≤ j < n。

示例 1：

输入：nums = [3,10,5,25,2,8]
输出：28
解释：最大运算结果是 5 XOR 25 = 28.

示例 2：

输入：nums = [14,70,53,83,49,91,36,80,92,51,66,70]
输出：127

提示：

  • 1 <= nums.length <= 2 * 105
  • 0 <= nums[i] <= 231 - 1

题解

既然提到了异或操作，那么肯定需要对每个数的二进制形式下功夫，因此此题需要用一种数据结构存储每个数的二进制形式。

那么怎么选呢，最简单的当然是分开存储，即建立一个数组，每一位一一对应nums数组的每一个数字的二进制形式，最后用一个嵌套循环找到异或的最大结果。

这种方法是最直观可以想到的，具体内容不过多阐述。

字典树(前缀树)

思考一下，既然我们要得到异或最大值，那是不是遍历找与某个数相异或为最大的数的时候，只要找当前位二进制相反的数即可

如图，想要找的是预期的最大值，但是nums中不一定恰巧存在，于是我们可以退而求其次，低位的可以与预期的不同，这样可以尽可能保证我们得到的异或值是最大的。

于是我们想在要做的便是当遍历这个数的时候从高位到低位，依次遍历每一个二进制数，并找是否有数可以使得异或值尽可能大。
